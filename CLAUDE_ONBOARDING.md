# Claude Onboarding

Before making any changes to this project, complete the following checklist.

## Step 1 - Read Documentation

Read these files completely:

- PROJECT.md
- DESIGN_SYSTEM.md
- CODING_STANDARDS.md
- ROADMAP.md
- README.md
- PROJECT_STATUS.md

## Step 2 - Inspect Project

Review the entire repository.

Understand:

- folder structure
- existing architecture
- reusable components
- current milestone

Do not assume anything.

## Step 3 - Report

Provide:

- Current project phase
- Completed milestones
- Missing milestones
- Files you inspected

Do not write code.

## Step 4 - Wait

Wait for the next instruction.

Never begin implementation until explicitly instructed.

## Rules

- Work only inside this repository.
- Never create another project.
- Never use a temporary workspace.
- Never remove existing files unless instructed.
- Save every file directly into this repository.
- Preserve architecture and coding standards.

## Before Finishing Any Task

Confirm:

- Every file was saved.
- List every created file.
- List every modified file.
- No files were accidentally deleted.
